# EEG-and-HRV-Integration-for-Cognitive-Stress-Assessment
R code for EEG and HRV analysis of cognitive stress dataset
This repository contains the Jamovi-generated R syntax used for the repeated
measures ANOVA and mixed-effects models reported in the manuscript. It can be
executed using the jmv and GAMLj3 packages in R. The code reproduces all
statistical results presented in the article.
