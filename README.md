# EEG-and-HRV-Integration-for-Cognitive-Stress-Assessment

R code for EEG and HRV analysis of cognitive stress dataset. This repository contains the Jamovi-generated R syntax used for the statistical analysis reported in the manuscript. 

1. Setup and Dependencies

It can be executed using the jmv and GAMLj3 packages in R. The code reproduces all statistical results presented in the article.

2. Dataset
The dataset analyzed in this study is publicly available on Kaggle at: https://www.kaggle.com/datasets/apithm/ecg-and-eeg-stress-features.
Ensure that the dataset is placed in the appropriate directory before running the scripts.
